## 🛠️ 使用教程 | How to Use

### Windows / Mac 用户 (推荐使用 Clash)
1. 下载并安装 [Clash Verge](https://github.com/clash-verge-rev/clash-verge-rev) 或其他 Clash 客户端。
2. 复制上方的 **Clash 订阅链接**。
3. 在客户端的 `配置 (Profiles)` 页面中，将链接粘贴至输入框并点击 `导入 (Download)`。
4. 选中刚刚下载的配置文件，在 `代理 (Proxies)` 页面选择延迟较低的节点。
5. 开启 `系统代理 (System Proxy)` 即可畅游网络。

### Android (安卓) 用户
1. 下载并安装 **v2rayNG** 或 **Clash for Android**。
2. 复制对应的订阅链接 (Base64 或 Clash 链接)。
3. 打开客户端，进入“订阅设置” -> “添加”，粘贴链接并保存。
4. 返回主界面，点击“更新订阅”，测试延迟后选择可用节点，点击右下角按钮连接。

### iOS (苹果) 用户
1. 在非国区 App Store 搜索并下载 **Shadowrocket** (小火箭) 或 **Quantumult X**。
2. 复制上方的 **Shadowrocket 订阅链接**。
3. 打开软件，点击右上角 `+` 号，`类型` 选择 `Subscribe (订阅)`。
4. 粘贴 URL 链接，点击保存。右滑配置文件点击“更新”，随后选择节点并开启顶部开关。
